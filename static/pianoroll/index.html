Getting started (local setup)
  1. In the repo root, run preflight (checks FFmpeg, FluidSynth, SoundFont):
    python scripts/beginner_preflight.py
  2. Start the API (or use uvicorn app:app):
    python scripts/beginner_launch.py
    Full steps: docs/BEGINNER_FIRST_RUN_CHECKLIST.md in the repo.
  3. Here: use Import audio or Record, then double-click a clip to edit.
  4. If something looks wrong: open /api/v1/health in a new tab.

Quick checklist (local)

  • Preflight (repo root): python scripts/beginner_preflight.py — checks FFmpeg, FluidSynth, and SoundFont.
  • Launch server: python scripts/beginner_launch.py (or uvicorn app:app from repo root).
  • In Studio: Import audio or Record, then double-click a clip to edit.
  • Verify the backend: /api/v1/health — JSON shows checks (soundfont, ffmpeg, fluidsynth).
Common issues
  • Missing SoundFont: add a .sf2 under assets/ (default assets/piano.sf2).
  • Port in use or bind error: stop other servers or change PORT in .env.
  • First pip install is slow: large ML/audio dependencies are expected.

The full checklist file lives in your repo clone: docs/BEGINNER_FIRST_RUN_CHECKLIST.md
