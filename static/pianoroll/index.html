How to edit (MVP)
1) Click empty grid: set cursor and choose a cell.
2) Press Insert Note: add a default note at the selected cell.
3) Click a note: select; drag to move; drag the right edge to resize.
4) Press Delete: delete selected note.
Cancel discards changes; Save writes back into the clip.
